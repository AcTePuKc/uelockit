# Subnautica 2 Turkish Localization Helper

## Prerequisites

Before using this helper, make sure you have:

- Python `3.10` or newer
- PowerShell
- `UnrealLocres`
- `UnrealPak`
- `retoc`

Put your extracted files here:

- `source/Game.en.json`
- `source/Game.en.locres`
- `source/Game.tr.nested.json`
- `source/Game.tr.locres`

`source/` is for extracted game files.

`working/` is for the editable translation files generated by the helper scripts.

Then:

1. Copy `config/UELocKit.sample.config.psd1` to `config/UELocKit.config.psd1`
2. Make sure Python `3.10` or newer is installed and available as `python`
3. Fill in your local tool paths
4. Run:
   - `powershell -ExecutionPolicy Bypass -File .\tools\refresh-working-tr.ps1`
5. Review or edit:
   - `working/Game.tr.json`
   - or `working/Game.tr.csv` if you prefer CSV
6. If the game updates, compare the new source strings:
   - `python .\tools\check_source_update.py --working .\working\Game.tr.json --source-json .\source\Game.en.json`
7. Build translated locres:
   - `powershell -ExecutionPolicy Bypass -File .\build-locres.ps1 -ConfigPath .\config\UELocKit.config.psd1 -TranslationFormat json`
8. Build standalone package:
   - `powershell -ExecutionPolicy Bypass -Command "& '.\tools\build-standalone-tr.ps1' -ConfigPath '.\config\UELocKit.config.psd1' -TranslationFormat json -CreateZip 1 -DeployToGame 0"`

Expected output:

- `pak-output/Subnautica2-TR_P.pak`
- `pak-output/Subnautica2-TR_P.utoc`
- `pak-output/Subnautica2-TR_P.ucas`
- `pak-output/Subnautica2-TR_P.zip`

This helper expects the Turkish translation JSON in the nested export format and converts it into a working row-based JSON automatically.

JSON is the recommended editing format.

CSV is also generated for translators who prefer spreadsheet-style editing.

This helper does not install Python automatically.
